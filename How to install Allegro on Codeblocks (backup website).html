Ambiente: Code::Blocks + Allegro 4

Code::Blocks com MinGW e Allegro 4 (Windows)

O presente documento descreve os procedimentos para instalar e configurar o ambiente de desenvolvimento Code::Blocks em Windows integrado ao conjunto de ferramentas MinGW e a biblioteca Allegro. As instruções que se seguem dizem respeito ao Windows XP e oferece algumas pistas específicas para o Windows Vista.

Etapa 1: Instalação do Code::Blocks com o MinGW integrado

  1. É recomendado baixar e usar a última versão estável com o MinGW incluso: clique “binary release” em http://www.codeblocks.org/downloads e selecione o arquivo codeblocks-*mingw-setup.exe, onde * representa o número da última versão estável.
  2. Execute, de forma padrão, o arquivo de instalação com exceção do diretório alvo. Altere o diretório de instalação para C:\CodeBlocks.
  3. Aguardue o término da instalação e prossiga para a próxima etapa.

Etapa 2: Configuração do MinGW

  1. Abra a tela “Propriedades do sistema” ao pressionar <Tecla Windows> + Pause/Break, selecione a aba “Avançado” (no Windows Vista, selecionar "Configurações avançadas do sistema", clicar na aba "Avançado" da caixa de diálogo "Propriedades do Sistema") e clique no botão “Variáveis do ambiente”.
  2. Uma vez aberta a caixa de diálogo “Variáveis de ambiente”, crie uma nova variável de usuário de nome MINGDIR. Para tal é necessário clicar no botão “Nova” do bloco “Variáveis de usuário” e digitar MINGDIR no campo de rótulo “Nome da variável”. A seguir digite C:\CodeBlocks\MinGW no campo “Valor da variável” e clique no botão “OK”.
  3. Selecione a variável Path no bloco “Variáveis do sistema” da caixa de diálogo “Variáveis de ambiente” já aberta, clique no botão “Editar” e insera no final do conteúdo do campo “Valor da variável”, um ponto-e-vírgula (;) seguido por C:\CodeBlocks\MinGW\bin. A seguir, clique no botâo "OK". 
  4. Abra o “Prompt de comando” (Iniciar > Todos os programas > Acessórios > Prompt de comando) e digite
    gcc -v
    para testar a instalação do MinGW. Se o resultado for a apresentação da informação referente à versão do compilador gcc instalado através do MinGW, então tudo está pronto para dar prosseguimento à próxima etapa.

Etapa 3: Instalação do Allegro

  1. Para baixar a última versão estável do Allegro vá  para http://www.allegro.cc/ e  selecione Files.
  2. Na seção Binaries, no item MinGW (http://static.allegro.cc/file/library/allegro-4.2.3/allegro-mingw-4.2.3.zip, na versão 4.2.3 do Allegro) para baixar o arquivo zip necessário para a instalação da biblioteca. As pastas bin, include e lib com os seus respectivos arquivos no zip devem ser extraídas para o diretório c:\CodeBlocks\MinGW\allegro. No final você deve ter uma pasta c:\CodeBlocks\MinGW\allegro\bin, uma c:\CodeBlocks\MinGW\allegro\include e uma c:\CodeBlocks\MinGW\allegro\lib.
  3. Baixe o arquivo alleg42.dll e o copie para o diretório Windows system32 [C:\Windows\System32] e siga para a próxima etapa.

Etapa 4: Configuração do Compilador e do Depurador no Code::Blocks

  1. Inicie a execução do Code::Blocks. Caso solicitado, opte pelo GNU GCC Compiler. Mantenha a associação do Code::Blocks com arquivos C e C++. Clique na opção “Settings” da barra de menus e selecione “Compiler and debugger...”
  2. Assim que abrir a caixa de diálogo “Compiler and debugger settings”, assegure-se de que a opção “GNU GCC Compiler” está selecionada em “Selected compiler”
  3. Clique na aba “Toolchain executables” e assegure-se de que o campo “Compiler's installation directory” contenha
    C:\CodeBlocks\MinGW
  4. Clique na aba “Search directories” e efetue as seguintes adições:
    1. Na aba subordinada “Compiler” de “Search directories”, adicione, nessa ordem, com o auxílio do botão "Add":
      C:\CodeBlocks\MinGW\include
      C:\CodeBlocks\MinGW\allegro\include
    2. Na aba subordinada “Linker” de “Search directories”, adicione, nessa ordem, com o auxílio do botão "Add":
      C:\CodeBlocks\MinGW\lib
      C:\CodeBlocks\MinGW\allegro\lib
    3. Na aba subordinada “Resource Compiler” de “Search directories”, adicione, com o auxílio do botão "Add":
      C:\CodeBlocks\MinGW\include
  5. Clique na aba “Linker settings” e adicione, nessa ordem, com o auxílio do botão "Add":
    C:\CodeBlocks\MinGW\lib\libgdi32.a
    C:\CodeBlocks\MinGW\allegro\lib\liballeg.a
  6. Clique no botão “OK” e vá para a próxima etapa.

Etapa 5: Modelos (templates) do Allegro

  1. Baixe o arquivo Allegro Code::Blocks Template e o descomprima em seu diretório C:\CodeBlocks\templates.
  2. Siga o modelo no subdiretório “Allegro Static Link” quando necessitar a capacidade de ligação estática do Allegro.
  3. Siga o modelo no subdiretório "Allegro Dynamic Link" quando necessitar a capacidade padrão de ligação dinâmica do Allegro. Neste caso o usuário de seu jogo precisa ter o arquivo alleg42.dll no diretório C:\WINDOWS para poder executar o código do jogo.

Etapa 6: Verificação final

Compile e execute os programas abaixo para verificar se o Allegro foi corretamente instalado.

#include <allegro.h>
int main(void)
{
    allegro_init();
    allegro_message("Hello World!");
    return 0;
}
END_OF_MAIN()

Debugging in Code::Blocks



Configurando Allegro no Code::Blocks no Linux

  • Abrir o Code::Blocks
  • Selecionar Settings > Compiler and Debugger > Linker Settings
  • Adicionar `allegro-config --libs` (inclusive os `) em "Other linker options"

Mappy Tile Map Editor

Para gerar fundos de uma tela de jogo, use o Mappy Tile Map Editor. Para instalá-lo, entre em http://www.tilemap.co.uk/mappy.php e baixe a última versão do software que foi desenvolvido para o Windows. Descomprima o conteúdo do arquivo baixado em C:\mapwin. Para usar o editor, clique no arquivo mapwin.exe no diretório em que foi descomprimido o arquivo baixado em http://www.tilemap.co.uk/mappy.php.
